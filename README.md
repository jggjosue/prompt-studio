# Visionary Vault

This is a [Next.js](https://nextjs.org/) project created in Firebase Studio.

## Getting Started

Follow these steps to get your development environment running.

### Install dependencies

Run the following command in the root of your project to install the necessary dependencies:

```bash
npm install
```

### Update Environment Variables

Create a `.env.local` file in the root of your project. You can copy the contents of the existing `.env` file to start.

### Run the Development Server

Once the dependencies are installed and your environment variables are set, you can run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see your application.
